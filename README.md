# VITYARTHI – Student Record Management System

## Overview
VITYARTHI is a simple and efficient Student Record Management System built using Python.  
It runs entirely through the command line and stores student information using nested dictionaries, allowing quick and clean CRUD operations — Create, Read, Update, and Delete records.

This project is suitable for beginners learning Python, especially topics like data structures, functions, and menu-driven programming. The entire system works in memory, making it lightweight and easy to run without extra setup.

## Features
- Add new student records  
- Search for an existing student  
- Update stored student details  
- Delete any student entry  
- Display all records in a formatted table  
- Fast lookups using nested dictionaries  
- Simple, modular, beginner-friendly code  
- Entire system contained in a single Python file  
- No external dependencies (no CSV, JSON, or database required)

## Technologies / Tools Used
- Python 3.8 or above  
- Standard Python library only (no installations required)

## Installation and Running the Project

1. Download or clone the project folder  
2. Ensure Python 3 is installed
